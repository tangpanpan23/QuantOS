# QuantSaaS 开发路线图

> 最后更新：2026-05-02

---

## 📊 项目愿景

首个将**宏观政策感知**、**AI 辅助决策**与**用户自主创造**深度融合的量化策略 SaaS 平台。

**核心价值：**
- 🎯 技术民主化 - 让机构级量化能力惠及专业投资者
- 🤖 AI + 人工协同 - AI 负责分析，人负责决策
- 📊 全链路服务 - 数据感知 → 策略构建 → 回测验证 → 实盘执行

## 🗺️ 开发阶段

### Phase 0 - 基础建设 ✅ (已完成)

**目标：** 搭建完整的 Go + go-zero 框架

- [x] go-zero REST API 框架
- [x] JWT 用户认证系统
- [x] MySQL 数据库 + GORM AutoMigrate
- [x] Redis 缓存层
- [x] Docker Compose 开发环境
- [x] Kubernetes 生产部署
- [x] Proto RPC 接口定义（预留）
- [x] 完整 API 路由注册（60+ 端点）
- [x] 所有 Handler Stub 实现（可编译）
- [x] 文档体系（API / Developer / Setup / Changelog）

### Phase 1 - 核心功能 🟡 (进行中)

**目标：** 让平台可以实际使用

#### 1.1 策略管理（40%）
- [x] 策略 CRUD 路由 + Stub
- [ ] `createStrategyHandler` - 完整实现（保存到 DB）
- [ ] `getStrategiesHandler` - 完整实现（分页 + 筛选）
- [ ] `runStrategyHandler` - 策略回测触发
- [ ] 策略模板系统
- [ ] 策略市场（分享/克隆）

#### 1.2 市场数据 API（60%）
- [x] `stock_screener.py` - Python 数据脚本
- [x] 涨停池 / 强势股 / 板块龙头
- [x] 技术指标计算（MA / RSI / MACD）
- [ ] `getKLineDataHandler` - 对接 Python 脚本
- [ ] `getRealTimeQuoteHandler` - 实时行情
- [ ] `getStockListHandler` - 全市场股票列表
- [ ] WebSocket 实时推送

#### 1.3 新闻政策感知（0%）
- [ ] 多源新闻接入（东方财富、同花顺）
- [ ] NLP 情感分析
- [ ] 政策事件提取
- [ ] 影响传导图谱
- [ ] `getNewsHandler` - 新闻列表
- [ ] `getPolicyHandler` - 政策专题

#### 1.4 投资组合（30%）
- [x] 投资组合路由 + Stub
- [ ] 完整 CRUD 实现
- [ ] 持仓管理
- [ ] 绩效计算
