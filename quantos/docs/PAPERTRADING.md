# 模拟盘交易系统 (Paper Trading)

> Tank 专属量化交易模拟系统，基于 QuantOS 平台

## 📋 概述

本模块提供完整的模拟盘交易功能，用于：
1. **历史回测**：模拟半年交易，验证策略可行性
2. **日常交易练习**：手动记录交易，熟悉交易流程
3. **策略验证**：检验选股信号和风控规则

---

## 🚀 快速开始

### 1. 安装 Go（如果还没有）

```bash
# macOS amd64 (Intel)
cd /tmp
curl -L -o go.tar.gz https://go.dev/dl/go1.22.5.darwin-amd64.tar.gz
tar -C ~/go -xzf go.tar.gz
export PATH=$PATH:~/go/go/bin
go version
```

### 2. 构建

```bash
cd ~/Code/QuantOS/quantos
export PATH=$PATH:~/go/go/bin
GOPROXY=https://goproxy.cn,direct go build -o console ./app/command/
```

### 3. 运行

```bash
# 查看账户状态
./console paper status

# 运行历史回测
./console paper simulate

# 买入
./console paper buy 600036 35.50 300 "MA多头+RSI=45"

# 卖出
./console paper sell 600036 38.50 "止盈"

# 重置账户
./console paper reset
```

### paper buy
买入股票

```bash
./console paper buy <代码> <价格> <股数> <理由> [-n 名称] [-r RSI值]
```

示例：
```bash
# 买入招商银行
./console paper buy 600036 35.50 300 "MA多头+RSI=45" -n "招商银行"

# 买入长江电力
./console paper buy 600900 22.50 600 "价格突破MA20" -n "长江电力"
```

参数说明：
- `代码`：股票代码（必须）
- `价格`：买入价格（必须）
- `股数`：买入股数，必须是100的倍数（必须）
- `理由`：买入理由（必须）
- `-n`：股票名称（可选，默认用代码）
- `-r`：RSI值（可选，默认50）

### paper sell
卖出股票

```bash
./console paper sell <代码> <价格> [理由]
```

示例：
```bash
# 手动卖出
./console paper sell 600036 38.50 "止盈"

# 止损
./console paper sell 600036 34.30 "止损"
```

### paper check
检查持仓的止损止盈条件

```bash
./console paper check [代码:价格 ...]
```

示例：
```bash
# 检查所有持仓
./console paper check

# 提供当前价格检查
./console paper check 600036:35.50 600900:22.10
```

### paper trades
查看交易记录

```bash
./console paper trades [-l N]
```

示例：
```bash
# 查看最近20条
./console paper trades

# 查看最近50条
./console paper trades -l 50
```

## 📊 交易规则

### 账户配置

| 参数 | 值 | 说明 |
|------|-----|------|
| 初始资金 | ¥100,000 | 总资金 |
| 可交易资金 | ¥60,000 | 扣除保留现金 |
| 保留现金 | ¥40,000 | 始终保留不动用 |

### 仓位控制

| 参数 | 值 | 说明 |
|------|-----|------|
| 单只最大持仓 | ¥15,000 | 每只股票最多投入 |
| 最大持仓数 | 4 只 | 同时最多持有4只 |
| 最小交易单位 | 100 股 | A股最小买卖单位 |

### 止损规则

```
🛑 止损线：亏损达到 3% 立即卖出
计算公式：止损价 = 买入价 × (1 - 3%)
```

### 止盈规则

```
💰 第一止盈：盈利达到 5% → 卖出 50%
💰 第二止盈：盈利达到 8% → 卖出剩余 50%
```

### 移动止损

```
📈 当盈利 > 5% 后，止损线移动到成本价（+0.5%缓冲）
目的：锁定部分利润，避免亏损
```

### 🚨 铁律（绝对不能违反）

```
🚨 单日亏损 > ¥2,000 → 停止当天交易
🚨 单只亏损 > ¥1,000 → 立即止损
🚨 持仓已达 4 只 → 不能买入新股票
🚨 大盘下跌 > 2% → 停止买入（观察是否要清仓）
```

## 📈 回测分析

### 运行回测

```bash
./console paper simulate
```

回测会：
1. 模拟半年交易（2025-10-16 到 2026-04-15）
2. 每5天检查一次买入信号
3. 实时检查止损止盈条件
4. 模拟结束时强制平仓

### 解读回测结果

关注以下指标：

| 指标 | 说明 | 目标 |
|------|------|------|
| 收益率 | 总资产相对初始资金的百分比 | > 0% |
| 胜率 | 盈利交易 / 总交易数 | > 50% |
| 均盈/均亏比 | 平均盈利 / 平均亏损 | > 1.5 |
| 最大连续亏损 | 连续亏损最多的金额 | < ¥5,000 |

## ❓ 常见问题

**Q: 买入时提示"资金不足"？**
A: 检查可用资金是否足够买入（价格 × 股数），保留现金¥40,000不可用

**Q: 买入时提示"持仓已满"？**
A: 最多同时持有4只股票，需要先卖出才能买入新的

**Q: 数据保存在哪里？**
A: `data/paper/` 目录下，JSON格式，可直接查看

**Q: 如何清空所有数据重新开始？**
A: 运行 `./console paper reset`
